## Gist 복사, 다운로드, 내장하기

Gist는 아이디어를 저장하거나 친구와 공유하기 위해 사용할 짧은 코드 조각입니다.

Gist의 진짜 힘은 실제 Git 저장소라는 사실에서 옵니다. 이 말은 어떤 gist를 원래 저자가 아니더라도 clone 또는 fork할 수 있습니다. 또한 gist를 블로그나 자바스크립트를 지원하는 Internet-y 텍스트 필드에 내장할 수 있습니다.
